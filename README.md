# GarpixBooking

## Description

Тестовое приложение для бронирования номеров в отеле.

Используется бэкэнд: https://github.com/Dmit0/booking.

Посмотреть работу можно на стенеде: https://testers.help:8877/

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```
